# AI Trip Planner

AI Trip Planner is a web application that helps users plan their trips efficiently. It provides personalized travel suggestions, destinations, and itineraries using AI integration.

---

## Features

- Search for destinations using Google Places API  
- Personalized trip recommendations  
- Dynamic itineraries with travel suggestions  
- Responsive design for mobile and desktop  
- Simple and intuitive user interface

---

## Tech Stack

- **Frontend:** React, HTML, CSS, JavaScript  
- **Backend:** Node.js, Express.js  
- **APIs:** Google Places API  
- **Other Tools:** Axios for API calls, Tailwind CSS for styling
